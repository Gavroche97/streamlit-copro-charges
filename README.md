# Streamlit Copro Charges

This project is a Streamlit application designed to calculate the annual copropriété charges for a residence based on provisions and proposed quotes.

## Project Structure

```
streamlit-copro-charges
├── app.py                # Main entry point of the Streamlit application
├── src
│   ├── ui.py            # User interface components
│   └── utils.py         # Utility functions for data processing
├── data
│   └── sample_provisions.csv  # Sample data for testing
├── requirements.txt      # Python dependencies
├── .streamlit
│   └── config.toml      # Streamlit configuration settings
└── README.md             # Project documentation
```

## Installation

To run this application, you need to have Python installed on your machine. You can install the required dependencies using pip:

```
pip install -r requirements.txt
```

## Running the Application

To start the Streamlit application, run the following command in your terminal:

```
streamlit run app.py
```

## Usage

Once the application is running, you will be able to input provisions and quotes to calculate the annual copropriété charges. The user interface will guide you through the process.
